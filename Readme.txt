1. How to build the project

- Please set "PathFinding" project as a starter project and build with x64 configuration / Debug or Release mode.
  Or you can execute with "PathFinding.exe" file in x64\Debug(or Release) folder.

- GitHub Link:
https://github.com/kanious/Project01_SteeringBehavious



2. User Input Option

- You can see all options in the UI in game.

< For Camera >
* WASD : Move Camera
* QE : Rotate Camera (90 degree per pressing)
* Mouse wheel drag : Zoom In/Out

< For Player >
* Mouse LB Click : Set a goal tile
                  (You cannot set a goal while the player is moving)
* F1 : Start to move the player to the goal tile

< Etc >
* ESC : Close the program.



3. Simple Description

< Tile Information >
* Black : Wall (The player cannot pass this tile)
* White : Floor (The player can pass this tile)
* Green : Start Position
* Red : Goal Position
* Blue : The tiles connecting the start tile and goal tile
